
jma-info
=====

入れ方
-----

適当なフォルダに`./jma-info`フォルダと`./jma-info.rb`を入れればOK

メモ
-----

- get_nameメソッドを使っていたり使っていなかったりしている
- 表示部分だったりをクラスで纏めたい
- 構文エラーで落ちないように(get-infoのメイン部分をrequireにする)
- get-infoでネットアクセスに失敗したときにデータが失われる

### 残り

- 火山関連
- 東海地震
- 竜巻注意
- 土砂災害
- 指定河川洪水予報
- 紫外線観測データ
- 降灰予報（定時）
- 警報級の可能性
- 台風情報
- 海上予報
- 季節予報
